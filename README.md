# Installation Instruction
**Author**: `Sanskar Bhusal` <br>
**Project**: Expense Tracker

## Installation Instruction
### Install Elysia (Bun.js)
1. clone the repository
```bash
git clone git@github.com:sanskarbhusal/expense-tracker.git
```
2. Install packages 

```bash
cd expense-tracker/backend
bun install
```
3. Start the bun server
```bash
bun start
```
4. You can then access the API at: <br>
[http://localhost:3000](http://localhost:3000)

### Install React (Vite + Node.js)
1. Spin up another terminal session and navigate to frontend folder.
```bash
cd expense-tracker/frontend
```
2. Install packages 

```bash
npm install
```
3. Start the vite server
```bash
npm run dev
```
4. You can then access the frontend at: <br>
[http://localhost:5173](http://localhost:5173)
